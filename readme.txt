FSXPilot 3.3 32 bit Version

The 32 bit version is produced for backwards compatibility with FSX. 
- Please unzip this in a directory of your choice. 
- Start FSX. 
- Run fsxpilot.exe as an administrator. 

MSFS2020: FSXPilot 64 bit is required. 

XPLANE: You need to start FSXPilot via the included XPLANE plugin.
XPLANE 11: FSXPilot 32 bit will not work. The FSXPilot 64 bit version is required.
